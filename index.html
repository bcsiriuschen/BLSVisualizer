BLSVisualizer is a visualization tool used to view time series data imported from the Bureau of Labor Statistics. The tree view organizes the data in a hierarchical order, making it easy to navigate to the desired dataset. The user can drag and drop the desired data into the chart slots, causing the corresponding plot on the right to automatically update with the newly added data. The user may add multiple time series data to each plot and can remove them individually by clicking on them in the plotting box or all of them by clicking "Clear". Orange icons indicate dimensional values, whereas green icons indicate rates.
